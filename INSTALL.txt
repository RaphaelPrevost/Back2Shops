
== Dependency ==
=== For user.backtoshops.com ===
1. Pillow dependency(Details reference: https://pypi.python.org/pypi/Pillow/2.1.0):
   * Ubuntu 12.04 LTS or Raspian Wheezy 7.0:
     $ sudo apt-get install libtiff4-dev libjpeg8-dev zlib1g-dev libfreetype6-dev liblcms1-dev libwebp-dev

   * Max OS X
     $ brew install libtiff libjpeg webp littlecms

2. install gevent / redis
   * Ubuntu 12.04 LTS or Raspian Wheezy 7.0:
     $ sudo apt-get install libevent-dev redis-server

   * Max OS X
     $ brew install libevent redis

3. install swig
   * Ubuntu 12.04 LTS or Raspian Wheezy 7.0:
     $ sudo apt-get install swig

   * Max OS X
     $ brew install swig

4. Add pip config to install M2Crypto
   $ cd ~/.pip
   $ touch pip.conf
   Add following config into pip.conf (change the path to your real path, e.g. /home/jessica/svn/backtoshops/packages/dist)
    {{{
    [global]
    find-links =
        /path/to/backtoshops/packages/dist

    [install]
    find-links =
        /path/to/backtoshops/packages/dist
    }}}
